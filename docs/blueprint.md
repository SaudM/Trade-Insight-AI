# **App Name**: Trade Insights AI

## Core Features:

- Daily Trade Logging: Log trades with details like time, symbol, direction, position size, entry/exit reasons, P/L, and emotional state.
- Daily AI Analysis: Generates a concise daily report summarizing trading activity, identifying strengths and weaknesses (focusing on process), and analyzing the impact of emotional state. Uses reasoning as a tool for assessing which details to include.
- Weekly Pattern Discovery: Analyzes the week's trading logs to find recurring error patterns and successful strategies. Assesses position sizing effectiveness and correlates emotional states with trading outcomes.
- Weekly Improvement Plan: Generates a "Weekly Improvement Plan" with actionable steps based on the weekly pattern analysis, using AI reasoning.
- Monthly Performance Review: Compares current and previous months' trading habits and performance. Highlights persistent issues in the trading system and evaluates strategy execution. Uses AI as a reasoning tool to suggest improvements.
- System Iteration Suggestions: Provides specific, actionable recommendations for improving the trading system based on monthly analysis. Leverages AI pattern identification to help you build a consistent and repeatable trading strategy. AI serves as a reasoning tool for drawing connections across all insights.
- Data Visualization: Showcase interactive charts and graphs visualizing the trading data. This will focus on visualizing P/L, trade frequency, emotion influence, win/loss ratios, etc.

## Style Guidelines:

- Primary color: Navy blue (#2E3A87), evokes professionalism and trust.
- Background color: Light gray (#F0F4F8), provides a clean, neutral backdrop.
- Accent color: Teal (#339E93), provides contrast and highlights key elements.
- Headline font: 'Space Grotesk', sans-serif. Body font: 'Inter', sans-serif.
- Use clear, minimalist icons to represent data and actions.
- A clean and organized layout that facilitates quick data entry and review.
- Subtle transitions and animations to provide feedback on user actions.
